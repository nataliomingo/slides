2. Calculating the partition function and the Helmholtz free energy¶

We calculate the partition function including the configurational and magnetic degrees of freedom. From it, the Free energy is: $F = - $kB$ T ln Z$. Use partition functions calculated above and divide by formUnits to get $F(N\text{Co}, T=50-2550 \text{ K})$ with and without magnetic entropy.

In [4]:
import math
from numpy import log as ln

def make_F_T_array(ABC_df_sorted,T):
    k_B = 1.38064852e-23
    eV_to_joule = 1.60218e-19
    print('calculating T = ',T)
    boltz_T = ABC_df_sorted['form_E'].apply(lambda E : math.exp(-(E*eV_to_joule)/(k_B*T)))
    ABC_df_sorted['boltz_T_scaled'] = boltz_T*ABC_df_sorted['scaling_factor']
    ABC_df_sorted['boltz_T_scaled_mag'] = boltz_T*ABC_df_sorted['scaling_factor']*ABC_df_sorted['mag_states']
    # Groupby Co_count and sum to get partition functions
    Z_T_config_only = ABC_df_sorted.groupby(['Co_count'])['boltz_T_scaled'].sum()
    Z_T_config_mag = ABC_df_sorted.groupby(['Co_count'])['boltz_T_scaled_mag'].sum()
    # Use partition functions calculated above and divide by formUnits to get F(Co_count, T) 
    # with and without magnetic entropy.
    F_T_config_only = -k_B*T*ln(ABC_df_sorted.groupby(['Co_count'])['boltz_T_scaled'].sum())/ (formUnits*eV_to_joule)
    F_T_config_mag = -k_B*T*ln(ABC_df_sorted.groupby(['Co_count'])['boltz_T_scaled_mag'].sum())/ (formUnits*eV_to_joule)
    return F_T_config_only, F_T_config_mag

Calculating gaps¶

1. Let's define the functions we need¶

a) Function to interpolate $F(T,x)$:¶

This is important, since we need smooth functions whose derivates we can calculate, in order to determine coexistence curves.

In [7]:
# Function to innterpolate the configurational and magnetic free energies
from scipy import interpolate
def interpolate_F(Fconfonly,Fconfmag,temperatures,**ss):
    Fc = np.array([F.to_numpy() for F in Fconfonly])
    Fm = np.array([Fcmag.to_numpy() - Fcon for Fcon,Fcmag in zip(Fc,Fconfmag)])
    x = np.arange(0,Fc[0].size)
    y = np.array(temperatures)
    [a,b] = np.meshgrid(x,y)
    wght=np.meshgrid([1]+[0 for ii in range(23)]+[1],y)
    fcc = interpolate.bisplrep(a, b, Fc,w=10*np.ravel(wght[0])+1.,**ss)    
    fmm = interpolate.bisplrep(a, b, Fm,w=10*np.ravel(wght[0])+1.,**ss)    
    return fcc, fmm

c) Function to determine the coexistence regions (i.e. solubility gaps)¶

At a given temperature $T$, the solubility gap is the range of concentration values $x$ between two convex "protrusions" of the free energy curve at $x_1$ and $x_2$, such that \begin{eqnarray} \frac{\partial F}{\partial x}\big |_{x_1}=\frac{\partial F}{\partial x}\big|_{x_2} = \frac{F(x_1)-F(x_2)}{x_1-x_2}. \text{ (1)} \end{eqnarray} We solve these two nonlinear equations using scipy.optimize.fsolve. To prevent the resulting $x_1$ or $x_2$ from getting out of the $0<x<N_\text{Co}$ range, we express them in terms of another function, $y(p)$, which is bounded between 0 and 1: \begin{eqnarray} x_1 = y(p_1),\text{ } x_2 = y(p_2),\text{ } y(p)\equiv \frac{N_\text{Co}}{1+e^{-p}}, \end{eqnarray} and we solve equations (1) to find $p_1$ and $p_2$.

In [49]:
from scipy.optimize import fsolve
import math
def CDE_gap_bounded(fcev,fcev_p,temps,numx,*attributes):
    yyfunc = lambda x:0.0+(numx-0.0)/(1.+np.exp(-x))
    mygap =[]
    initsearch = (-1.,2) #these initial values need to be tuned by hand, to guarantee finding the solutions
    for T in temps:
        def ff(p): 
            y1,y2 = yyfunc(p[0]),yyfunc(p[1])
            return(lambda x1,x2: 
                   ((fcev(T,x1) - fcev(T,x2))/(x1-x2) - fcev_p(T,x1),(fcev(T,x1) - fcev(T,x2))/(x1-x2) - fcev_p(T,x2)))(y1,y2)
        gap = fsolve(ff,initsearch,factor=.1)
        initsearch = gap
        mygap = mygap + [[yyfunc(gap[0]),yyfunc(gap[1])]]
        plt.plot(np.arange(0,numx,.01),[fcev(T,xii) for xii in np.arange(0,numx,.01)])
        plt.plot([yyfunc(gap[0]),yyfunc(gap[1])],[fcev(T,xii) for xii in [yyfunc(gap[0]),yyfunc(gap[1])]])
    plt.show()
    plt.plot(np.array(mygap)[:,0],temps)
    plt.plot(np.array(mygap)[:,1],temps)
    return mygap
Let's see how this function works¶

First, interpolate the free energy. fcev is the free energy with only configurational entropy contributions, whereas fev contains both configurational and magnetic entropy contributions. We also calculate their first derivative with respect to $x$, fcev_p and fev_p.

d) Function to correct the configurational free energy near the composition edges¶

This function corrects the entropy, replacing the 24 atom combinations, $\log(N! / n! (24 - n)!$, by the inifite atom combinations, $N \log(N/e) - n \log(n/e) - (24 - n) \log((24 - n)/e)$. Its importance will become clear later.

In [57]:
from scipy.special import gamma as gammaf
def corrector(nn):
    d=0.0001
    N = 24
    n = nn*(1-2.*d/N)+d
    aa = 1000.
    return (N*np.log(aa*N) - n*np.log(aa*n) - (N-n)*np.log(aa*(N-n))) / np.log(gammaf(N+1)/(gammaf(n+1)*gammaf(N-n+1)))

The difference with the uncorrected one is not noticeable in this case. But it will become crucial to understand the experimental solubility gap:

2. Solubility gaps with separate curves for two phases¶

The experimental solubility gap cannot be explained using only the curves above. This is clear when realizing that there is a tetragonal to cubic phase transition for the pure Mn$_3$O$_4$ phase, at a temperature $T_c\sim 1443\text{K}$. We need to consider a free energy curve $F(T,x)$ consisting of the lowest free energy amongst the two competing phases.

We do not have an ab initio $F_\text{cub}$ curve for the cubic phase. But we know a few things about it:

  • At $T=0,x=0$, the cubic phase's free energy is much above the tetragonal one's: $F_\text{cub}-F_\text{tetr}=3.8$eV, calculated with DFT [CHECK THIS VALUE!!].
  • For $T=T_c,x=0$, the two free energies have the same value: $F_\text{cub}-F_\text{tetr}=0$.
  • When $T>T_c,x=0$, the cubic phase is the stable one, so $F_\text{cub}<F_\text{tetr}$.
